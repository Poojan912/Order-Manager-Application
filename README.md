# Order Manager Application

## Overview

Welcome to the Order Manager Application! This Android application is developed using Kotlin and Jetpack Compose in Android Studio. It is designed to help businesses manage orders efficiently and gain insights into their sales performance through detailed charts and analytics.

## Features

### Order Management
- **Customer Orders**: Maintain records of all orders placed by customers. Each order entry includes essential details such as order ID, customer information, order date, items ordered, and total amount.

### Sales Analytics
- **Monthly Sales Charts**: Visualize monthly sales data through interactive charts. These charts provide insights into the number of orders and total revenue for each month.
- **Yearly Sales Charts**: Analyze yearly sales trends to understand long-term business performance. Yearly charts aggregate monthly data to present a comprehensive view of sales over the years.

### User Authentication
- **Login and Signup**: Secure login and signup functionality using Firebase Authentication. Users can create accounts and log in using their email and password.
- **Password Recovery**: Allow users to recover their password through Firebase's password reset feature, ensuring access to their accounts.

### User Interface
- **Jetpack Compose UI**: The application is built with Jetpack Compose, ensuring a modern and responsive user interface. The UI components are designed to be intuitive and user-friendly, providing a seamless experience.

## Technologies Used
- **Kotlin**: The primary programming language used for developing the application.
- **Jetpack Compose**: A modern toolkit for building native Android UI, enabling fast and efficient UI development.
- **Firebase Authentication**: For secure user authentication and management.
- **MVVM Architecture**: The application follows the MVVM (Model-View-ViewModel) architecture pattern to ensure clean separation of concerns and maintainable code.
- **Material Design**: The UI follows Material Design principles, providing a consistent and visually appealing look and feel.

## Getting Started
1. **Clone the Repository**: 
    ```bash
    git clone https://github.com/Poojan912/Order-Manager-Application
    ```
2. **Open in Android Studio**: Open the project in Android Studio to start working with the code.
3. **Firebase Configuration**: Add your Firebase project configuration files to the app to enable authentication features.
4. **Build the Project**: Sync the project with Gradle and build it to ensure all dependencies are properly configured.
5. **Run the App**: Deploy the app on an emulator or a physical device to start managing orders, viewing sales analytics, and using authentication features.
